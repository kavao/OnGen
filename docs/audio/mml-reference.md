# MMLリファレンス

TL;DR: OnGen の MML は PPMCK/MCK 流の音長指定（`C4` = 四分音符の C）と相対オクターブ（`>` / `<`）を正本とします。

## 音長とオクターブ

音符直後の数字は**音長**を表します。オクターブは `O` コマンド、または `>`（1 つ上げる）/ `<`（1 つ下げる）で指定します。

```text
PPMCK / MCK: c4 = 四分音符のC
OnGen:       C4 = 四分音符のC（オクターブは O / > / < で指定）
```

旧バージョンでは音符直後の数字をオクターブとして解釈していましたが、現在は廃止されています。

## コマンド一覧

| コマンド | 意味 |
|---------|------|
| `C D E` | ノート（音長省略時は L コマンドの値） |
| `C4 D8 E16.` | ノート + 音長指定（付点も併用可） |
| `O4` | オクターブ指定 |
| `>` / `<` | オクターブを 1 つ上げる / 下げる |
| `L8` | デフォルト音長 |
| `T120` | テンポ (BPM) |
| `V15` | 音量 (0〜15) |
| `R` / `R8` | 休符 |
| `C&C` | タイ（直前の音符と音長を連結） |
| `[CDE]3` | ループ（任意回数の反復。数字省略時は 2 回） |
| `@1` | 音色プリセット（1〜7） |
| `%12` | FM 変調指数 (0〜15) |
| `*4` | FM モジュレータ倍率（値÷2） |
| `~8` | LFO 深度 (0〜15) |
| `W(kick.wav)` | WAV サンプル再生 |

独自拡張の `%`、`*`、`~`、`W(...)` は PPMCK/MCK 記法ではありません。

一致しないトークンは `ValueError` としてパーサーエラーになります（サイレントスキップしません）。

## PPMCK / MCK 互換性

| PPMCK / MCK記法 | 現状 | 備考 |
|-----------------|------|------|
| `c d e f g a b` | 部分対応 | 大文字・小文字を同一視 |
| `+`, `-` | 対応 | シャープ・フラット |
| `r4`, `r8` | 対応 | 休符の音長指定 |
| `t120` | 対応 | テンポ |
| `l4`, `l8` | 対応 | デフォルト音長 |
| `o4` | 対応 | オクターブ指定 |
| `v0`〜`v15` | 対応 | 固定音量 |
| `c4`, `c8` | 対応 | 音符ごとの音長指定 |
| `>`, `<` | 対応 | 相対オクターブ変更 |
| `c.`, `c4.` | 対応 | 音長指定と付点の組み合わせ |
| `&` | 対応 | タイ・音長連結 |
| `^` | 未対応 | タイの別表記（`&` を使用） |
| `q`, `@q` | 未対応 | ゲートタイム／クオンタイズ |
| `[ ... ]4` | 対応 | `]` 直後の数字で反復回数 |
| `|: ... :|` | 未対応 | 多重リピート |
| `K<num>` | 未対応 | トランスポーズ |
| `n<num>` | 未対応 | ノート番号直接指定 |
| トラック文字 `A`〜`E` | 未対応 | CLI の `--track` / `--track-file` を使用 |
| `#TITLE` 等のヘッダ | 未対応 | `ValueError`（未対応トークンとしてエラー） |
| `@v`, `@EN`, `@EP` 等のマクロ | 未対応 | エンベロープ・アルペジオマクロ |
| `@DPCM` | 未対応 | 代替として `W(...)` を使用 |

対応状況のロードマップは `_workingspace/plans/mml-compatibility-roadmap.md` にあります。

## 実行例

```bash
python tools/sound/sfx_generator.py --input "O4 L4 T120 C D E G" --style square -o output/mml/melody
```

ABC 記法は [BGMと楽譜](bgm-and-scores.md) を参照してください。
