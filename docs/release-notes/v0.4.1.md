# OnGen v0.4.1

MML / ABC からチップチューン風の効果音・BGM を生成する CLI ツールです。実装正本は `tools/sound/sfx_generator.py`（MIT License）。

## ハイライト

- **Pyxel MML 方言** — `--mml-dialect pyxel` で Pyxel 2.4+ 向け MML を試聴
- **Pyxel マルチパート** — `--pyxel-multipart` で 1 行 1 パートの Composer 形式をミックス
- **MML 効果音サンプル** — `scores/sfx_*_sample.mml`（ジャンプ・コイン・打撃・爆発など）
- **実曲サンプル** — ナハトムジーク冒頭 4 小節、Pyxel Composer 由来 4 パート曲
- **合成の安定性** — フィルター後の波形クリップでノイズ系プリセットのクリッピングを防止

## クイックスタート

```bash
pip install -r requirements.txt
python tools/sound/sfx_generator.py --preset coin -o output/sfx/coin
python tools/sound/sfx_generator.py --input-file scores/sfx_jump_sample.mml -o output/sfx/jump
python tools/sound/sfx_generator.py --mml-dialect pyxel --pyxel-multipart \
  --input-file scores/pyxel_composer_sample.mml -o output/pyxel/composer
```
